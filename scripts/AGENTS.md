# Scripts Guide

This folder contains local start and stop scripts for the Dockerized MVP.

## Scripts

- `start.sh`: starts the Docker Compose app on Mac and Linux.
- `stop.sh`: stops the Docker Compose app on Mac and Linux.
- `start.ps1`: starts the Docker Compose app on Windows PowerShell.
- `stop.ps1`: stops the Docker Compose app on Windows PowerShell.
- `start.bat`: starts the Docker Compose app on Windows Command Prompt.
- `stop.bat`: stops the Docker Compose app on Windows Command Prompt.

Keep scripts small wrappers around Docker Compose unless the project needs more setup later.
